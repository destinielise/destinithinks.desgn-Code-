Roads to Revenue

web design

Opportunity

During my time at AdsNative, I was introduced to a printed piece the company commissioned to educate publishers. This brochure featured not only information but also offered solutions that only AdsNative could solve. To further expand on this idea, I created a digital interactive map.

The map did not appeal to all
of our potential clients

The printed map was not as approachable to a customer new to native advertising. We needed to educate first-time buyers and still inform publishers about our product.
 

Delicate Details

I wanted the map to be fully interactive without being javascript heavy. All the map elements were masked to allow the full exploration of Roads to Revenue.

Coded with Love

The site was coded using a combination of HTML and Javascript. I used my knowledge of developing to write the code for the project.  Each element is absolute positioned using percentages to allow flexibility.

Roads to Revenue is a great tool to allow customers to learn more about our company. However, the since the map was content heavy it didn't distill to mobile very well. To push this project further, I would explore different web languages to add mobile friendly interactions.
